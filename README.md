# Bases heatmap view

Calendar-style heatmap visualization for any date-based property in your Bases tables. Track activity, habits, progress, or any numeric/date-related data with beautiful color-coded squares.

This plugin is based on [cookiecutter-obsidian-plugin](https://github.com/serafinovsky/cookiecutter-obsidian-plugin/). [First steps after install](https://serafinovsky.github.io/cookiecutter-obsidian-plugin/).
