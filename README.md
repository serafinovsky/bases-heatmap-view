# Bases heatmap view

![Heatmap visualization example](assets/heatmap-example.png)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Obsidian](https://img.shields.io/badge/Obsidian-1.10.0+-7c3aed?logo=obsidian)](https://obsidian.md)

**GitHub-contributions-style heatmap** for any date-based property in your **Bases** tables. Track activity, habits, and progress with color-coded squares.

## How to use

1. Configure the filter to show the records you want
2. Create a new view and select **Heatmap**
3. Select the property to display on the map and adjust other settings
4. You're all set

![Preview](assets/preview.gif)
